# Customer-Shopping-Experience-Excel-Dashboard
Interactive Excel dashboard analyzing customer shopping behavior by region, age group, gender, product category, and order status to generate business insights.

Customer Shopping Experience Dashboard

This Excel dashboard analyzes customer shopping behavior based on:
- Region
- Gender
- Age Group
- Product Category
- Order Status

Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

The dashboard helps identify top regions, categories, and cities by sales
and understand customer purchasing patterns.
